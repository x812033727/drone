# 40-1 認證路線圖(台灣/美國/歐盟)

> 策略:Phase 0–2 以原型與試點為主(在各地「非認證即可操作」的框架內飛行),Phase 3 正式投入型式認證。但**設計階段就把認證需求做進去**(Remote ID、燈光、標識、日誌、降落傘介面、EMC 裕度),避免後期重設計。
> 本文為工程規劃視角的摘要,執行前需以最新法規原文與顧問確認(法規更新頻繁)。

## 1. 台灣(民航局 CAA + NCC)

| 主題 | 要點 | 對我們的影響 |
|------|------|--------------|
| 法源 | 民用航空法遙控無人機專章 + 遙控無人機管理規則 | — |
| 重量分級 | 250 g / 1 kg / 2 kg / 15 kg / 25 kg 為關鍵門檻;**25 kg 以上須逐案核准與檢驗** | PA-1(6 kg)走註冊 + 操作證;PB-1(48 kg)需申請檢驗合格證,流程長——排 Phase 3 |
| 操作限制 | 400 ft 以下、視距內、日間;禁航區/限航區圖資;夜間/BVLOS/人群上方需專案申請 | 巡邏夜航需專案核准;物流 BVLOS 依「無人機物流專案」個案申請(已有業者先例) |
| 操作人 | 普通/專業操作證(法人需專業證) | 試點客戶培訓納入產品交付 |
| 頻率 | NCC 低功率電波(2.4/5.8 GHz LP0002)、5G 模組電信審驗 | 射頻模組選已認證品可簡化 |
| 產品檢驗 | 25 kg 以上機型性能與耐航檢驗(類 mini 型式認證) | PB-1 的主要法規工作量 |

## 2. 美國(FAA)

| 主題 | 要點 | 對我們的影響 |
|------|------|--------------|
| 商用操作 | Part 107(< 55 lb / 25 kg):操作員考照即可商用 | PA-1 完全落在 Part 107 內,美國市場首選機型 |
| Remote ID | 強制 Standard Remote ID(整機廣播)——製造商需向 FAA 提交 Declaration of Compliance(DoC) | **製造商義務**:RID 模組 + 測試報告 + DoC,PA-1 出貨美國前必辦 |
| BVLOS | 目前靠 waiver;Part 108(BVLOS 常態化)規則制定中 | 物流業務排美國後期;跟蹤 Part 108 進度 |
| > 25 kg | 需型式認證(Type Certification,Durability & Reliability 路徑)——成本極高 | PB-1 美國市場僅農噴走 Part 137(農噴豁免路徑,44807),物流不排 |
| 射頻 | FCC ID(數傳)、模組預認證(蜂窩) | 同左 |
| 供應鏈 | NDAA Section 848 / American Security Drone Act:政府客戶禁中國關鍵零件 | 飛控/圖傳/相機供應鏈履歷從 Phase 1 建立,是核心賣點 |

## 3. 歐盟(EASA)

| 主題 | 要點 | 對我們的影響 |
|------|------|--------------|
| 分類 | Open(低風險)/ Specific(中)/ Certified(高) | PA-1 目標 Open A2 或 Specific;PB-1 屬 Specific |
| C 級標章 | 製造商需取得 C 級(C0–C6)產品認證(通知機構驗證) | PA-1 目標 **C2 標章**(< 4 kg... 注意:C2 為 < 25 kg 但有距離規則)+ CE(RED、EMC、機械、電池) |
| Specific 類 | 操作人提交 SORA 風險評估;標準場景 STS-01/02 可簡化 | 農噴/物流在歐盟走 SORA;提供客戶 SORA 技術資料包是產品配套 |
| Direct RID | C1 以上強制 | 已內建 |
| 資安 | CRA(Cyber Resilience Act)將適用聯網產品 | SBOM、OTA 簽章、漏洞通報流程從 Phase 1 記帳 |

## 5. 認證時程與費用量級(Phase 3 預算依據)

| 項目 | 時程 | 費用量級(NTD) |
|------|------|----------------|
| NCC + 台灣 CAA(PA-1) | 3–6 個月 | 0.5–1 M |
| FCC + FAA RID DoC(PA-1) | 3–6 個月 | 1–2 M |
| CE + C2 標章(PA-1,通知機構) | 9–15 個月 | 3–6 M |
| 台灣 25 kg+ 檢驗(PB-1) | 6–12 個月 | 1–3 M |
| 電池 UN38.3(每 pack 型號) | 2–3 個月 | 0.3–0.5 M |

順序建議:台灣(母市場、成本低)→ 美國(Part 107 門檻低、NDAA 順風)→ 歐盟(C 標章最重,放最後)。
