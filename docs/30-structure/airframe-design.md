# 30-1 機體結構設計

## 1. PA-1(四軸摺疊,軸距 680 mm)

### 構型與佈局

```
        前
   M1 ↗     ↖ M2          M1-M4:馬達位(X 構型)
     \       /
      [機身艙]             機身艙(上下夾板式):
     /       \               上層:GNSS 天線、數傳天線
   M4 ↘     ↙ M3            中層:FC-H7(隔震)、Jetson、5G
        後                    下層:電池倉(抽拉式)、酬載快拆
```

- **X 四軸、臂向下摺疊**:摺疊後 430 × 420 × 260 mm,單箱手提
- 機身艙三明治結構:碳纖維板(上下)+ CNC 鋁合金角件 + 碳管臂(Ø25 mm)
- 電池倉後置抽拉 + 酬載前下掛:電池(2.1 kg)與酬載(≤1.5 kg)對 CG 前後配平;CG 容差設計 ±15 mm,任何合法配置落在容差內
- 天線佈局:GNSS 天線最高點(碳纖維遮蔽故天線外置)、數傳天線於腳架(下視野)、5G 天線機身側面,彼此隔離 > 15 cm
- 腳架固定式(高度預留雲台視野);Phase 2 評估收放腳架(全景巡邏需求)

### 關鍵結構件

| 件 | 材料/製程(原型) | 備註 |
|----|------------------|------|
| 機身上下板 | 3K 碳纖維平紋板 CNC 銑切 2.5 mm | 開孔減重 + 走線槽 |
| 機臂 | 捲製碳管 Ø25×1.5 mm | 摺疊鉸鏈 CNC 鋁 7075 |
| 摺疊鉸鏈 | CNC 7075 + 不鏽鋼銷 + 快鎖 | 專利檢索!(DJI 摺疊機構專利密集區) |
| 馬達座 | CNC 6061 | 含 ESC 散熱面 |
| 電池倉/滑軌 | 射出 PC+ABS(原型 CNC POM) | 盲插電極 + 二段鎖 |
| 隔震 | FC 隔震球 + IMU 子板灌膠 | 兩級隔震 |

## 2. PB-1(六軸摺疊,軸距 1650 mm)

- 六臂等角佈局,臂 Ø40 mm 碳管、雙節摺疊(向後收攏),摺疊後 900 × 780 × 500 mm
- 中央主框架:碳板 + 鋁型材混合(承受 20 kg 酬載動載),酬載艙位於 CG 正下方——藥箱/貨箱互換不移 CG
- 電池倉左右對稱雙倉(頂部),熱插拔滑軌 + 高電流盲插(Amass/安費諾 300A 級)
- 農噴構型:噴桿沿臂佈置(噴頭於槳下洗流區,提升穿透)、全機線束防藥液(套管 + 灌膠接頭)、可整機水洗(IP55)
- 物流構型:貨箱快拆 + 降落傘艙(頂部,彈射傘,觸發來自飛控 + 獨立氣壓計)
- 起落架:固定寬距(藥箱裝卸空間),滑撬式,底部耐磨橡膠

## 3. 結構設計準則(兩平台)

| 項目 | 準則 |
|------|------|
| 安全係數 | 飛行載荷 ×1.5(極限)/ ×2.0(關鍵接頭);地面粗暴搬運工況另算 |
| 剛度 | 一階臂彎曲模態 > 3× 槳頻(避免共振);以 FEA + 敲擊試驗驗證 |
| 振動 | 馬達座至 IMU 傳遞率 < 0.3(20–200 Hz) |
| 疲勞 | 摺疊機構 5,000 次開合;快拆介面 2,000 次插拔 |
| 跌落/運輸 | 包裝跌落 80 cm;摺疊態車載振動 2 h(ISTA 標準) |
| 熱設計 | ESC/Jetson 傳導至結構散熱;45°C 環境滿載懸停 30 min 無降頻 |
| 防水 | PA-1 IP43(縫隙迷宮 + 疏水膜透氣閥);PB-1 IP55(密封圈 + 灌膠) |

## 4. 開發流程

1. 概念佈局(CAD:Fusion 360/SolidWorks)+ CG/慣量表(與飛控調參共用)
2. FEA(靜載 + 模態)→ 3D 列印驗證裝配 → CNC 原型
3. 整機振動/熱測試 → 迭代
4. Phase 2 DFM:高複用件開模(射出/模壓),CNC 保留低量件
5. 產出物進 repo `structure/`:STEP、圖紙、FEA 報告、測試報告
