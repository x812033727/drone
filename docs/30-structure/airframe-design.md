## 4. 開發流程

1. 概念佈局(CAD:Fusion 360/SolidWorks)+ CG/慣量表(與飛控調參共用)
2. FEA(靜載 + 模態)→ 3D 列印驗證裝配 → CNC 原型

4. Phase 2 DFM:高複用件開模(射出/模壓),CNC 保留低量件

## 5. PA-1 佈局尺寸與站位(rev 2)

> 座標系(全文件與 §6/§7 共用):原點 = 軸距幾何中心(四馬達軸對角交點)於**中層板面**;x 前正、y 右正、z 上正(mm)。與 PX4 NED 的換算(x_NED=x、z_NED=−z)在調參文件層處理。佈局假設:艙體夾板 ~200×170 mm、艙高 ~120 mm(z −60…+60)、**夾板對角至摺疊鉸鏈軸半寬 90 mm**;腳架著地面 z ≈ −180(§1 雲台視野預留)。此組尺寸為 rev 2 凍結草案,CAD(Phase 1)收斂後回寫。

### 5.1 槳尖干涉檢查

| 項 | 數值 | 推導 |
|----|------|------|
| 對角馬達距(= 軸距) | 680 mm | 00-overview 規格 |
| 鄰臂馬達距 | **480.8 mm** | 680 ÷ √2 |
| 槳直徑(18") | 457.2 mm | propulsion §2 選型 18×6.1 |
| **鄰槳槳尖間隙** | **23.6 mm ≈ 5.2% D** | 480.8 − 457.2 |
| 槳尖至艙體 | ~21 mm | 槳尖內緣半徑 340 − 228.6 = 111.4,艙體對角半寬 ~90 |

- **判定:可行(邊際)**。業界經驗慣例(非標準,需查證)鄰槳間隙 ≥ 5–10% D:穿越/工業四軸常見 0.05–0.15 D,DJI M300 RTK 由公開幾何反推 ~0.19 D(寬裕取向)。本設計 5.2% 貼慣例下限:幾何上不干涉(臂端氣動撓度 < 0.1 mm、鉸鏈間隙致槳尖擺動 ±1–2 mm,間隙仍 > 20 mm),但槳尖渦互擾有 **2–4% 懸停效率損失風險**——列 W12 推力台「單軸 vs 對拉」對比實測項,若損失 > 3% 回饋 propulsion §4 效率假設
- **19" 上限口徑修正**:19" = 482.6 mm > 鄰臂馬達距 480.8 mm,**共面佈置下幾何干涉(−1.8 mm)**;propulsion §2「軸距 680 對角可容 18–19"」的 19" 選項需錯層(上下臂面交錯)或加大軸距才成立 → 列 §8 缺口清單交 M2 修正口徑。**共面佈置的槳徑上限即 18"**

### 5.2 摺疊包絡可行性(430 × 420 × 260 mm,18" 摺疊槳)

- 摺疊自由度:臂繞鉸鏈軸(z +40 處、半徑 90 mm)於「臂軸鉛垂面」內下折角 θ;摺疊槳兩葉繞槳夾銷收攏貼臂(單葉長 ≈ (457.2 − 轂 ~65)/2 ≈ **196 mm < 臂長 250 mm**,貼臂可容 ✓)
- 關鍵幾何:臂沿**對角線**方向下折,包絡可用半寬取箱體對角 √(215² + 210²) ≈ 300 mm(四角收納),非側邊 215 mm:
  - 水平伸展:90 + 250·cosθ + 馬達半徑 ~35 ≤ 300 → **θ ≥ ~46°**
  - 垂直下探:摺疊態頂面取上層板 +45(GNSS 天線桿快拆/摺倒,列 §8 缺口)→ 底限 z ≈ −215;馬達組最低點 40 − 250·sinθ − ~30 ≥ −215 → **θ ≤ ~63°**
- **判定:可行域存在(θ ≈ 50–60°,四角斜下收),典型 θ=55° 時側向裕度 ~30 mm、底部裕度 ~25 mm**——裕度中等偏緊,摺疊槳葉須朝鉸鏈側向上貼臂收攏;CAD 全干涉檢查(含線束/腳架)列 Phase 1 必辦,若不可行的備選 = 下折 + 後掠複合鉸鏈,或包絡高度 260 → 280(犧牲手提箱規格,M2 決)

## 7. 臂結構核算(rev 2)

### 7.1 幾何、斷面與材料

- **懸臂長 L**:馬達軸半徑 340 − 鉸鏈軸半寬 90(§5 佈局)= **250 mm**(敏感度掃至 280;夾持段另 +50 mm 入管長)
- **端部集中質量 M**:馬達 MN505-S 195 g + 槳含夾 50 g = **245 g**(同源:propulsion §3);+馬達座 ~30 g 作敏感度上界(ESC 已移臂根,§5.3)
- 材料:高模量捲製碳管,軸向彎曲模量 **E = 90 GPa(設計基線,需查證 + 來料抽測三點彎)**;保守下界 60 GPa 作敏感度;ρ = 1.6 g/cm³
- 斷面(Ø30×2.0 rev 2 基線):I = π(30⁴−26⁴)/64 = **17,330 mm⁴**,m̄ = 0.281 kg/m(Ø25×1.5 rev 1:I = 7,674 mm⁴,m̄ = 0.177 kg/m)

### 7.2 一階彎曲模態 vs 槳頻

裸懸臂公式 f₁ = (1.875²/2π)·√(EI/(m̄L⁴));端部集中質量修正(等效質量法):

**f₁ = (1/2π)·√( 3EI / ((M + 0.2427·m̄L)·L³) )**

槳頻(激振源):MN505-S KV320 + 18×6.1 懸停轉速由推力係數反推(T = C_T·ρ·n²·D⁴,C_T ≈ 0.09–0.11 級):MTOW 懸停 1.5 kg/軸 → **n ≈ 3,000–3,300 rpm → 1P = 50–55 Hz**;全包線(爬升/抗風)至 ~4,000 rpm = 67 Hz;2P(雙葉通過頻率)懸停 100–110 Hz。**需查證:T-Motor 未公開 MN505-S 配 18×6.1 成表(官網僅標 max 推力 6.6–6.7 kg 配建議槳 20–22",2026-07 查核),W12 推力台實測轉速定案**。

**§3 剛度準則門檻:f₁ > 3 × 懸停槳頻(1P)= 150–166 Hz**

### 7.3 判定與管徑掃描(E = 90 GPa、L = 250、端質量 245/275 g)

| 斷面 | f₁(Hz) | 4 臂重量(管長 0.30 m)| Δ重量 | 對門檻 150–166 Hz |
|------|---------|----------------------|-------|--------------------|
| Ø25×1.5(rev 1) | **96–115**(L 250–280) | 212 g | 基準 | **✗ 差 ~30%,且撞 2P 懸停帶 100–110 Hz(雙重不可)** |
| Ø25×2.0 | ~128 | 277 g | +65 g | ✗ |
| Ø30×1.5 | 144–152 | 258 g | +46 g | △ 僅貼下緣(回退選項) |
| **Ø30×2.0(rev 2 基線)** | **161–170** | 337 g | **+125 g** | **✓ 成立(邊際)** |
| Ø35×2.0 | 205–216 | 398 g | +186 g | ✓ 寬裕(超額備選) |

- **判定:Ø25×1.5 對「f₁ > 3× 槳頻」不成立 → rev 2 臂管升級 Ø30×2.0 高模量,f₁ ≈ 161–170 Hz ≥ 150–166 Hz,成立(邊際)**。三個放行閘(任一不過即升 Ø35×2.0 或 M2 議決準則細化):① 碳管 E ≥ 90 GPa 來料抽測(E = 60 GPa 時 f₁ 僅 132–139 Hz ✗);② 端部質量控制 ≤ ~275 g(馬達座減重設計);③ W12 實測懸停 ≤ ~3,300 rpm。**整臂含鉸鏈敲擊試驗**(§3 準則既有要求)為最終仲裁——鉸鏈/快鎖是剛度鏈最弱環,接頭柔度可拉低實測 f₁ 20–40%,本節剛性根部假設即因此偏不保守,鉸鏈鎖止面預緊設計為 Phase 1 重點
- **重量影響回饋(對 propulsion §3)**:+0.10–0.13 kg(視夾持段),結構列 0.46 → ~0.57–0.59,**超出 propulsion §3 成長準備金 0.10 kg → 依該表規則觸發減重行動檢視**;首選補償 = ESC 轉單 APD(−0.14 kg,propulsion §8 轉單條件 (2) 之重量超支情境),次選 = 腳架/GNSS 桿減重;本文件不私改 propulsion §3(單一事實來源),交 M2 與週更同場回寫 → 列 §8 缺口清單

### 7.4 靜強度快核(剛度主導確認)

全推力 ×1.5:臂端 4.5 kg × 1.5 = 66.2 N → 根部彎矩 16.6 N·m → σ = M·c/I = 16,550×15/17,330 ≈ **14.3 MPa**,對捲管彎曲強度 ~400–600 MPa(需查證)SF > 25——**臂設計由剛度(模態)主導,強度非約束**;真正的強度熱點在鉸鏈/夾持段(碳管壓潰,§1 內襯)與角件螺栓,歸 FEA(§7.5)
