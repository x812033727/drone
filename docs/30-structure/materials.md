# 30-2 材料與製程

## 1. 選材總表

| 材料 | 用途 | 選用理由 | 注意 |
|------|------|----------|------|
| 3K 碳纖維板(T300 級,2–3 mm) | 機身板、臂座板 | 比剛度最高;板材 CNC 原型友善 | 導電 → 天線需外置;切邊防潮封邊 |
| 捲製碳管(Ø25 / Ø40) | 機臂 | 成熟供應鏈(台灣現貨規格多) | 夾持處加內襯防壓潰 |
| 鋁 7075-T6 CNC | 摺疊鉸鏈、高應力接頭 | 強度接近鋼、重量減半 | 陽極處理防蝕(農噴環境) |
| 鋁 6061-T6 CNC | 馬達座、一般結構件 | 便宜、好加工、散熱 | — |
| PC+ABS 射出(原型 CNC POM / MJF 尼龍) | 電池倉、外殼、線蓋 | 韌性、量產成本低 | UV 安定劑(戶外) |
| PA12 + GF(MJF/SLS 3D 列印) | 原型與低量結構件 | 免模具、迭代快 | 各向異性,受力件慎用 |
| PP / PVDF / Viton | 藥箱、液路、密封(PB-1) | 耐農藥化學腐蝕 | 藥液相容性清單驗證 |
| EPDM / 矽膠 | 密封圈、隔震 | 耐候 | 壓縮永久變形選型 |

## 2. 原型 vs 量產製程對照

| 部件 | 原型(Phase 1,<10 台) | 量產(Phase 3,>200 台/年) | 轉換點效益 |
|------|------------------------|---------------------------|------------|
| 碳纖維板件 | 板材 CNC | 模壓(壓縮成型)一體件 | −40% 成本、+剛度、外觀 |
| 塑膠件 | CNC POM / MJF 列印 | 射出成型 | 單件 −80%(模具攤提後) |
| 鋁件 | CNC | CNC(高應力件維持)+ 壓鑄(蓋板類) | 視量 |
| 外觀殼 | 3D 列印 + 噴塗 | 射出 + 模內漆或噴塗 | 品牌質感 |
| 線束 | 手工 | 治具化外包 | 一致性(商用可靠度大戶) |

原則:**Phase 1 全部免開模**(CNC + 列印 + 現貨碳管),把迭代自由度留到設計凍結;Phase 2 結束時凍結結構,Phase 3 才投模具(單套射出模 30–80 萬 NTD,提前投模是新創常見的燒錢事故)。

## 3. 表面處理與防護

- 鋁件:硬質陽極(接頭)/ 一般陽極 + 封孔(農噴)
- 碳件:切邊環氧封邊;外觀件 UV 清漆
- PCB:全板 conformal coating(壓克力系,返修友善)
- 緊固件:不鏽鋼 A2 + 螺紋膠中強度;震動關鍵件加彈墊或 Nord-Lock
- 電接頭:鍍金 + 防水圈;農噴機接頭全部帶護套

## 4. 供應鏈備註(台灣優勢)

- 碳纖維板/管:台灣自行車與複材產業鏈現貨充足(嘉隆、拓凱體系周邊)
- CNC:台中精密加工聚落,打樣 1–2 週
- 射出與模具:量產階段本地開模具備成本與溝通優勢
- 此供應鏈組合天然符合 NDAA 供應鏈敘事,是對美銷售的加分項
