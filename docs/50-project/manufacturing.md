# 50-7 生產工序(骨架)

> rev 1 · 2026-07。整機生產的工位序、產測與校飛程序骨架。**全文為 Phase 1 DVT 前細化的骨架版**:工位節拍、治具圖面、通過準則數值於 EVT 實作經驗後定案。策略層(Make/Buy、代工 vs 自建、模具決策)見 [supply-chain.md](supply-chain.md),不重複;Phase 0 原型組裝按 [phase0/build-and-first-flight.md](phase0/build-and-first-flight.md) §1,不在本文範圍。

## 1. 與供應鏈策略的銜接

[supply-chain.md §5](supply-chain.md) Phase 3 決策樹的既定傾向:PCBA 與次組裝外包、**總裝/校準/產測自持**——本文即「自持段」的工序定義。無論屆時代工或自建,§3 燒錄金鑰注入與 §4–§5 校準校飛皆為自持紅線(機密性與飛行器特有工序,決策樹「傾向自建」欄的兩個理由)。

## 2. 總裝工位序

依「先靜後動、先結構後帶電」排序,每工位出站檢查通過才流入下一站:

| 工位 | 內容 | 出站檢查(骨架) |
|------|------|------------------|
| A1 結構 | 中心板/機臂/起落架組裝,扭力上膠按規範 | 扭力抽檢、外觀、緊固件標記漆 |
| A2 線束 | 治具化線束鋪設(供應商件,[supply-chain §1](supply-chain.md) 線束列)、端子到位確認 | 導通/絕緣測試(治具)、拉力抽檢 |
| A3 航電 | FC-H7/Jetson 載板/GNSS/數傳裝機,隔震件安裝 | 上電前檢查(對 build-and-first-flight §2 口徑產線化)、接地量測 |
| A4 動力 | 馬達/ESC/槳座安裝,動力線焊接或端子壓接 | 相序與轉向確認(未裝槳)、焊點/壓接抽檢 |

## 3. 韌體燒錄與金鑰注入工位

- **燒錄**:飛控 bootloader + 釋出韌體映像 + 對應參數包([firmware.md §7](../20-software/firmware.md) 綁定規則);Jetson 寫入釋出 rootfs 映像(A/B 兩分區同版)
- **金鑰注入**:每機裝置憑證與私鑰產生/燒錄(flash 加密區,[security.md §2](../20-software/security.md) PKI——私鑰不出裝置);憑證指紋 ↔ 機身序號綁定上傳 fleet-svc
- **管制**:本工位為機密性紅線(§1)——簽章/憑證簽發基礎設施不落產線,產線只持一次性簽發通道;操作雙人覆核,產出記錄(誰、何時、哪台機、哪個版本)全數歸檔
- 出站檢查:韌體版本/參數 hash 回讀比對、憑證鏈驗證、與測試閘道 mTLS 連線成功

## 4. 校準工位

| 項目 | 內容 | 備註 |
|------|------|------|
| 感測校準 | IMU 多面校準、加速度計水平、溫補資料載入(如有) | 治具轉台,Phase 1 DVT 前定治具規格 |
| 羅盤校準 | 帶殼帶槳全配重校準(產線遠離大型鐵磁物) | 干擾水準記錄入機歷 |
| RTK/GNSS | 天線相位中心偏移設定、開闊場收星驗證 | 與 §5 校飛可合站 |
| 酬載/雲台 | 雲台水平/歸中、相機-機體外參(測繪構型) | 依構型選作 |

## 5. 校飛程序(產測飛行剖面)

每台出廠機必飛,剖面固定、通過準則量化(數值 DVT 前以 EVT 機隊統計定案):

1. **繫留/低空懸停 2 min**:振動 RMS、姿態穩定度、電流平衡在包絡內(對 `ulog_report` 異常規則同源口徑)
2. **方形航線一圈(自動任務)**:航點追蹤誤差、RTK Fixed 保持率
3. **失效保護抽測**:RC 失聯觸發 RTL(每 N 台抽一台全項)
4. **判定**:校飛 ULog 跑產測版異常規則,零告警 + 目檢通過 → 出廠;任一不過 → 退 §4 或維修工位,重飛

校飛 ULog 隨機歷歸檔(與 [firmware.md §6](../20-software/firmware.md) 日誌規格同格式),是日後維保與事故調查的「出廠基線」。

### 5.1 機歷(每機生產紀錄)

每台機一份機歷,自 A1 開工即建檔、隨機出廠、與 fleet-svc 裝置登記綁定(序號為鍵):

- 各工位出站檢查結果與操作者、關鍵料件批號(電芯批次/馬達批次——追溯回收半徑用)
- §3 燒錄紀錄(韌體版/參數包版/憑證指紋)、§4 校準數據、§5 校飛 ULog 與判定
- 出廠後的維保事件由雲端維保單延續同一機歷(cloud-fleet §3 異常開單的追溯終點)

## 6. 產測治具清單(骨架)

| 治具 | 用途 | 對應工位 |
|------|------|----------|
| 推力台(產測版) | 動力總成抽測:推力/電流/振動對標定曲線 | A4 出站抽檢 |
| 線束測試治具 | 導通/絕緣/耐壓(漏電)自動測試 | A2 出站 |
| 電源/漏電測試 | 整機上電漏電流、各匯流排電壓紋波 | A3 出站 |
| RF 測試 | 數傳/蜂窩/GNSS 天線輻射與靈敏度抽測(暗室或屏蔽箱) | A3 出站抽檢 |
| 校準轉台 | IMU 多面校準 | §4 |
| 燒錄/注入治具 | 批量燒錄座 + 憑證簽發通道終端 | §3 |

產測系統與治具預算已列 Phase 3([budget.md §1](budget.md));Phase 1–2 以工程用治具(推力台、暗室租用,[02-verification-validation.md §6](../02-verification-validation.md))過渡。

## 7. 分階段落地

| 階段 | 落地項 |
|------|--------|
| Phase 1(EVT,<10 台) | 工序以 checklist 形式跑(工程師執行);校飛剖面初版與數據累積 |
| Phase 2(DVT/試點) | 本文細化為正式工序文件(SOP + 治具圖面);通過準則數值凍結 |
