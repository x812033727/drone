## 4. RFQ 詢價單模板(兼為 Phase 1 鋪路)

發給供應商的詢價單固定欄位如下;Phase 0 即對 Phase 1 供應商(T-Motor/Hobbywing 代理、電池 pack 廠、碳件 CNC)發出試探性 RFQ,W10 前回收比價(對 [README.md](README.md) 第 3 節並行工作線)。

| 欄位 | 說明 |
|------|------|
| 品項/規格 | 型號或圖面編號(CNC 件附 STEP) |
| 數量分級 | 2 / 10 / 50 / 200(取得量價曲線,供 Phase 1–2 預算) |
| 單價與幣別 | 含稅與否、匯率條款 |
| 交期 | 樣品交期 / 批量交期 |
| MOQ 與付款條件 | 最低訂購量、訂金比例 |
| 品質文件 | 出廠測試報告;電池另要求 UN38.3 能力聲明 |
| 有效期 | 報價有效天數 |

Phase 1 目標供應商欄(RFQ 收件人清單,持續補充):

| 類別 | 對象 | 備註 |
|------|------|------|
| 動力 | T-Motor 代理、Hobbywing 代理 | 對應 bom.md MN505-S / P80 III 選型 |
| 電池 | 台灣 pack 廠(ODM,含 BMS 協同設計) | 對 team.md 外包清單;要求 UN38.3 經驗 |

## 5. 驗收入庫與資產編號

- 到貨 48 小時內驗收:外觀 → 對單 → 上電/煙霧測試(電池量測內阻並記錄)
- 驗收紀錄與序號拍照歸檔;不良品當週退換
- **資產編號規則**:整機 `DEV-01`、`DEV-02`;子件掛整機編號(如 `DEV-01-FC`、`DEV-01-RTK`);共用地面設備 `GSE-xx`(基站、充電器、工具);電池獨立編號 `BAT-01`~`BAT-06` 並逐顆記錄循環數
- 兩機零件不混用不互換;維修換件須更新資產表並記入異常追蹤單
