# 50-2 預算估算

> 量級估算(NTD),假設台灣薪資水準與本地供應鏈;人力以「全薪 × 1.35(雇主成本)」計。用於融資規劃與現金流控管,採購/招募後滾動修正。

## 1. 各階段預算

### Phase 0(3 個月)— 約 NTD 3.5 M

| 項目 | 金額 |
|------|------|
| 人力(3 人 × 3 個月,創始工程團隊) | 1.8 M |
| 開發機兩套(X500+Pixhawk 6X+Orin+RTK+數傳)(採購明細見 [phase0/procurement.md](phase0/procurement.md)) | 0.6 M |
| 工作場地/工具/雜項 | 0.5 M |
| 招募與顧問 | 0.6 M |

### Phase 1(9 個月,6–8 人)— 約 NTD 18 M

| 項目 | 金額 |
|------|------|
| 人力(平均 7 人 × 9 個月) | 10.5 M |
| 硬體 NRE(PCB 三版、自研板、測試設備租用) | 2.9 M |
| 結構開發(CNC 多輪、3D 列印) | 1.5 M |
| 原型物料(PA-1 ×3 + 備品) | 1.2 M |
| 測試(環測、EMC 預掃、電池送測)與試飛場地 | 1.0 M |
| 雲端/軟體基礎設施 | 0.4 M |
| 保險、法務(專利檢索:摺疊機構!)、雜項 | 0.5 M |

### Phase 2(12 個月,10–15 人)— 約 NTD 42 M

| 項目 | 金額 |
|------|------|
| 人力(平均 12 人 × 12 個月) | 24 M |
| PA-1 小量產 15 台物料(×0.26 M) | 4 M |
| PB-1 原型 ×2 + 動力/電池 | 2.5 M |
| DVT 測試(可靠度、環測全項、HALT) | 3 M |
| 模具首批(高複用塑膠件) | 2.5 M |
| 試點支援(培訓、駐點、保險)(拆解見 [operations-support.md §7](operations-support.md)) | 2 M |
| 行銷/展會/認證顧問前期 | 2 M |
| 營運雜項 | 2 M |

### Phase 3(12 個月)— 約 NTD 35 M + 量產營運資金另計

認證(台美歐,見 [40-regulatory](../40-regulatory/certification-roadmap.md) 費用表約 6–12 M)、量產治具與產測系統、供應鏈備料、通路建置。

## 2. 累計資金需求

| 時點 | 累計投入 |
|------|----------|
| Phase 0 結束(2026 Q3) | ~3.5 M |
| Phase 1 結束(2027 Q2) | ~21.5 M |
| Phase 2 結束(2028 Q2) | ~63.5 M |
| Phase 3 結束(2029 H1) | ~100 M(部分由營收回補) |

→ 對應融資規劃:種子輪 25–30 M(cover 到 Phase 1 結束 + 緩衝)→ Pre-A/A 輪 60–80 M(Phase 2–3)。搭配政府資源降低股權稀釋:經濟部 SBIR/CITD、國防部/中科院合作案、無人機國家隊採購(台灣當前政策順風)。

## 3. 燒錢控制原則

1. 人力永遠是最大項(>55%)——寧可小團隊 + 外包非核心(工業設計、線束、pack、認證代辦)
2. 每階段退出條件不達,不啟動下一階段招募
3. 硬體採購里程碑化:rev B 驗證通過才下 rev C 物料
4. 收入提早:Phase 2 試點即收費(不做免費 POC),服務收入養研發
