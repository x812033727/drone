# 50-5 營運/售後服務

> rev 1 · 2026-07。本文件定義 PA-1 的交付、培訓、維保、RMA、保固與試點支援體系,承接 [01-requirements.md](../01-requirements.md) REQ-OPS 系列與 [00-overview.md §4](../00-overview.md) 商業模式(整機銷售 + 軟體訂閱,售後是續約率的地基)。法規義務以 [40-regulatory](../40-regulatory/certification-roadmap.md) 為準;試點期預算對應 [budget.md](budget.md) Phase 2 試點支援 2 M。

## 1. 產品交付包定義

| 包別 | 內容 |
|------|------|
| PA-1 標準包 | 整機(含摺疊槳 ×4)、遙控器、智慧電池 ×2、充電器、備品槳 ×2 對、運輸箱、快速指南(展開/收納對 REQ-OPS-01 單人 < 3 min) |
| 選配 | 酬載模組(測繪相機/雙光雲台,快拆介面)、RTK 基站、備用電池 |
| 合規資料包 | 註冊/操作證申請文件指引(台灣 CAA)、SORA 支援技術資料包(歐盟 Specific 類,見 [40-regulatory §3](../40-regulatory/certification-roadmap.md))、ULog 紀錄說明 |

## 2. 客戶培訓與取證輔導

- **交機培訓(1 天,含在售價)**:上午飛行操作(展開/校準/任務規劃/失效保護行為演練),下午維保(換槳/換電/檢查表/雲端平台操作)
- **CAA 專業操作證輔導**:法人客戶需專業操作證([40-regulatory §1](../40-regulatory/certification-roadmap.md)),提供考照教材、術科陪練與報名代辦轉介(不自營考場)
- **進階課(收費)**:測繪後處理工作流、農噴標定與藥液配置(PB-1 時擴充)

## 3. 維保體系(預防保養)

| 週期 | 項目 |
|------|------|
| 每 50 飛行小時 | 槳目視/緊固檢查、馬達軸承手感、線束與插件、電池外觀與 SoH 讀取 |
| 每 200 飛行小時 | 馬達更換評估、ESC 遙測回顧(溫度/轉速異常)、結構緊固全檢、感測器再校準 |
| 換季 | 防水膠條/密封檢查、韌體版本統一、備件盤點 |

更換門檻:槳受損即換;馬達累計 400 h 或遙測異常;電池循環 ≥ 400 次或 SoH < 80% 退役(引 [propulsion.md §7](../10-hardware/propulsion.md) 循環 ≥ 400 次 @80% 容量)。

**站點備件清單(REQ-OPS-02:現場免焊接換件 < 30 min)**:四大件——馬達(含臂組件)×1、槳 ×2 對、電池 ×2、遙控器 ×1;易耗——腳架、天線、螺絲包、密封件、SD 卡。

## 4. RMA 與維修流程

| 分級 | 適用 | 處置 |
|------|------|------|
| L1 現場自修 | 四大件與易耗(REQ-OPS-02 範圍) | 客戶依培訓自換,雲端開單記錄 |
| L2 寄修 | 航電/結構/雲台故障 | 寄回原廠,週轉機補位 |
| L3 換新 | 修復成本 > 60% 或安全性存疑 | 整機換新,舊機拆解分析回饋 FMEA |

- **週轉機制度(試點期)**:每試點客戶群配置週轉機 1 台,L2 寄修期間無縫接替
- **SLA 目標**:48 h 響應 / 7 天修復(**試點期承諾,量產前依維修數據修訂**)

## 5. 雲端維保聯動

- **自動異常開單**:落地後 ULog 自動上傳,log-svc 依規則(振動超標、電池衰減、EKF 告警)自動開維保單(引 [cloud-fleet.md §3](../20-software/cloud-fleet.md))
- **遠端診斷**:一律走 WireGuard 隧道 + 雲端雙人授權(引 [security.md §2](../20-software/security.md)),不開任何直連通道
- **韌體 OTA 節奏**:季度功能版 + 隨需安全修補;分批灰度推送、版本相容矩陣、可回滾(cloud-fleet OTA 管理,Phase 2 上線);S1/S2 相關修補優先強推

## 6. 保固政策

| 項目 | 條件 |
|------|------|
| 整機 | 12 個月 |
| 馬達/ESC | 6 個月或 200 飛行小時(先到者) |
| 電池 | 12 個月內 SoH ≥ 80%(BMS 紀錄為準),未達即換 |
| 除外條款 | 炸機/人為操作不當、藥液腐蝕(未依維保程序清洗)、私自改裝/拆解、超出包線飛行(ULog 為判定依據) |

## 7. 試點客戶支援模式(Phase 2)

對應 [budget.md](budget.md) Phase 2「試點支援(培訓、駐點、保險)2 M」拆解:

| 項目 | 內容 | 概算(NTD) |
|------|------|------|
| 培訓與取證輔導 | 3–5 家試點客戶交機培訓 + 操作證輔導 | 0.4 M |
| 駐點支援 | 飛測工程師輪駐(team.md Phase 2 編制 2 名),每客戶首月駐點、之後週訪 + 週報 | 1.0 M |
| 保險 | 試點機隊第三人責任險 | 0.4 M |
| 備件週轉 | 站點備件包 ×3 + 週轉機運轉費 | 0.2 M |

週報內容:飛行小時/架次、異常單彙總、客戶回饋 → 月度產品委員會(team.md §4)校準需求。

## 8. 事故響應流程

1. **停飛與保全**:涉事機立即停飛,ULog 與現場影像保全(禁止覆寫),雲端側鎖定該機日誌
2. **法規通報**:依 CAA 遙控無人機管理規則於規定時限內通報(致人傷亡/重大財損事件);美歐市場依當地規定
3. **保險理賠**:以 ULog + 維保紀錄為理賠依據(ULog 完整紀錄本為認證清單項,見 [40-regulatory §4](../40-regulatory/certification-roadmap.md))
4. **工程回饋**:事故或 S1/S2 缺陷(分級見 [02-V&V §7](../02-verification-validation.md))強制觸發 [03-safety FMEA 重審](../03-safety-analysis.md)(§8 事件觸發條款),結案前該機不復飛、S1 全機隊停飛
